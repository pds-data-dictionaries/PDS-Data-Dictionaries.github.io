PDS DD Stewards¶

PDS Data Dictionary Stewards are a group of PDS Discipline Node and IPDA personnel responsible for the maintenance of PDS4 Discipline Local Data Dictionaries.

Github Team: https://github.com/orgs/pds-data-dictionaries/teams/pds-dictionary-stewards Email List: pdsddstewards@jpl.nasa.gov

PDS DD Steward Admins¶

Sub-team of the PDS Data Dictionary Stewards responsible for managing the PDS Data Dictionaries Github Org.

Github Team: https://github.com/orgs/pds-data-dictionaries/teams/pds-dictionary-steward-admins
