Request a New LDD¶

Contact the dictionary stewards group (TBD email link) to discuss the creation of a new dictionary.

Request a New Class or Attribute¶

To request a new attribute or class to an LDD, create a new issue in the PDS4 LDD Issues Repo <https://github.com/pds-data-dictionaries/PDS4-LDD-Issue-Repo/issues/new/choose>.

Report a Bug¶

To report a bug found in either an LDD or the PDS Common namespace, create a new issue in the PDS4 LDD Issues Repo.
